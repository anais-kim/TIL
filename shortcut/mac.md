Mac Keyborad Shortcuts
=======================

- Command ⌘
- Shift ⇧
- Option ⌥
- Control ⌃
- Caps Lock ⇪
- Fn

---------------
### Basic
- Screenshot: Command⌘ + Shift⇧ + 1/2/3/4

----------------
### Finder
- Show Hidden Files: Command⌘ + Shift⇧ + .

---------------
### Chrome
- Switch User: Command⌘ + Shift⇧ + M
- Switch Window: Command⌘ + Tilde~
